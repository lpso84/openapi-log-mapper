# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it privately.

- Contact: luis.oliveira@example.com
- Subject: [SECURITY] openapi-log-mapper vulnerability
- Include: steps to reproduce, impact, and any relevant logs/screenshots

We will acknowledge receipt within 72 hours and provide a remediation plan or status update within 7 days.

## Scope

This policy applies to the code in this repository and the deployed service at https://openapi-log-mapper.vercel.app.

## Expectations

- Do not publicly disclose the issue before we provide a fix or mitigation plan.
- Do not attempt to access other users' data, the Vercel project settings, or GitHub admin functions.
- Only test against your own data and accounts.

Thank you for helping keep this project secure.
